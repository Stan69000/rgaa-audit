Fonctionnalités

Tout ce qu'il faut pour
un audit sérieux

Un outil modulaire — utilisez ce dont vous avez besoin, ignorez le reste.

🧩

Extension Chrome

Audit en contexte directement dans votre navigateur. Aucune installation lourde. Résultats en temps réel avec panel overlay.

⚙️

CLI Playwright

Automatisation complète depuis le terminal. Audit multi-pages, intégration CI/CD, rapports JSON / HTML / CSV.

⌨️

Simulation humaine

Navigation au clavier, zoom 200%, désactivation CSS, test responsive 320px — comme le ferait un vrai auditeur.

🤖

Analyse IA (opt-in)

Priorisation intelligente des non-conformités via Claude API. Utilisez votre propre clé — zéro coût d'infrastructure.

🔒

100% local

L'audit tourne dans votre navigateur ou sur votre machine. Aucune donnée envoyée à un tiers sans votre accord.

📖

Open-source MIT

Code source intégralement disponible sur GitHub. Contributions bienvenues — règles, reporters, intégrations.

Fonctionnement

8 étapes,
8 secondes

Le simulateur reproduit le comportement d'un auditeur RGAA humain de bout en bout.

  • 01

    Chargement de la page

    Avec attente de stabilisation du DOM — compatible avec les SPA React/Vue/Angular.

  • 02

    Audit DOM automatique

    13 thèmes RGAA scannés : images, couleurs, structure, formulaires, navigation…

  • 03

    Navigation clavier (Tab ×15)

    Chaque élément focusé est analysé — outline CSS, visibilité, ordre de tabulation.

  • 04

    Zoom 200% + sans CSS

    Détection de débordements, vérification de la lisibilité sans feuilles de style.

  • 05

    Test responsive 320px

    Redimensionnement à 320px de large — standard mobile WCAG.

  • 06

    Screenshots automatiques

    Captures à chaque étape clé pour documenter l'audit.

  • 07

    Calcul du taux de conformité

    Score RGAA officiel : conformes / (conformes + non-conformes).

  • 08

    Rapport + analyse IA (opt-in)

    Export HTML/JSON/CSV. Recommandations priorisées si clé Anthropic fournie.

Installation

Prêt en
3 minutes

CLI Node.js ou extension Chrome — choisissez selon votre usage.

⚙️
CLI (recommandé)
Audit automatisé, CI/CD, rapports
$ git clone https://github.com/Stan69000/rgaa-audit
$ cd rgaa-audit/cli && npm install
$ npx playwright install chromium
# Lancer un audit
$ node bin/rgaa-audit.js https://mon-site.fr -o html -s rapport.html
🧩
Extension Chrome
Audit en contexte, sans config
# 1. Télécharger le dossier extension/
$ git clone https://github.com/Stan69000/rgaa-audit
# 2. Dans Chrome
# chrome://extensions/
# → Mode développeur ON
# → Charger extension non empaquetée
# → Sélectionner le dossier extension/
