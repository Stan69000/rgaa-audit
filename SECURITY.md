# Security Policy

Merci de ne pas divulguer publiquement une faille avant correction.

## Signaler une vulnerabilite

Pour signaler un probleme de securite, ouvrez une issue privee ou contactez le mainteneur via le canal officiel du projet:

- Repository: https://github.com/Stan69000/rgaa-audit

Merci d'inclure:

- description du probleme,
- impact estime,
- etapes de reproduction,
- version concernee (`extension` et/ou `cli`),
- proposition de correctif si disponible.

## Delais cibles

- Accuse de reception: sous 7 jours ouvrables.
- Qualification initiale: sous 14 jours ouvrables.
- Correctif: selon severite et complexite.

## Bonnes pratiques pour les utilisateurs

- Utiliser la derniere version disponible.
- Eviter d'executer des builds non verifies.
- Limiter les permissions de l'extension au strict necessaire.
